Der Code in "Paper 3 Smooth Q Learning Model 1" und "Paper 3 Smooth Dyna-Q Model 1" ist fast identisch zu "Paper 3 Collusion with Tabular Q Learning Model 1" mit dem Unterschied, dass
"Paper 3 Smooth Q Learning Model 1" nun den smoothing algorithmus beeinhaltet und "Paper 3 Smooth Dyna-Q Model 1" zudem noch Planning beinhaltet und dem Algorithmus jeweils ein Model des Marketes übergibt. 
Ansonsten sind Handling und Beschreibungen der Codes identisch zu "Paper 3 Smooth Q Learning Model 1". Für weitere Informationen verweise ich auf eben genannte Readme Datei.
